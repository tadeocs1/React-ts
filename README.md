# Sistema de Gestión de Pólizas

Aplicación web desarrollada con React y APIs REST para la administración de pólizas y clientes.

## 🚀 Funcionalidades
- Registro de pólizas
- Consulta de pólizas
- CRUD completo
- Consumo de API

## 🛠️ Tecnologías
- React
- TypeScript / JavaScript
- API REST
